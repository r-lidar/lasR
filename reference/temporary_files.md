# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpCjgXWU/file220c29334cfa.shp"
templaz()
#> [1] "/tmp/RtmpCjgXWU/file220c2e132fe5.laz"
```

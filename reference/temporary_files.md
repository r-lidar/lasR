# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpKyaJIW/file248b14b8b14d.shp"
templaz()
#> [1] "/tmp/RtmpKyaJIW/file248b503ac51a.laz"
```

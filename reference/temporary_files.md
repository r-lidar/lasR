# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpXXWUgg/file21b8533849eb.shp"
templaz()
#> [1] "/tmp/RtmpXXWUgg/file21b838954902.laz"
```

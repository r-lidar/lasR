# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpUYStHU/file24332d823e01.shp"
templaz()
#> [1] "/tmp/RtmpUYStHU/file243325390134.laz"
```

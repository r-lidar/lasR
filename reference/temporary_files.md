# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmp6nzY6f/file2465680ff883.shp"
templaz()
#> [1] "/tmp/Rtmp6nzY6f/file24657f4999e8.laz"
```

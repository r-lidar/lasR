# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpd9DKfI/file2443692d2bea.shp"
templaz()
#> [1] "/tmp/Rtmpd9DKfI/file24434f0aecb0.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpq3MByp/file24086867b9f9.shp"
templaz()
#> [1] "/tmp/Rtmpq3MByp/file24083f43d1d9.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpedO4G2/file23d72b25a7e9.shp"
templaz()
#> [1] "/tmp/RtmpedO4G2/file23d73d625d9a.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmppsKGHt/file22421e45fdf7.shp"
templaz()
#> [1] "/tmp/RtmppsKGHt/file22427421fd95.laz"
```

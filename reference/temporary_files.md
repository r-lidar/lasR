# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpzf9FCo/file22793a667e54.shp"
templaz()
#> [1] "/tmp/Rtmpzf9FCo/file22799ffe505.laz"
```

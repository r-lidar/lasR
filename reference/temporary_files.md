# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpo09ej6/file220e3663c4bd.shp"
templaz()
#> [1] "/tmp/Rtmpo09ej6/file220e211a283.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpQD0Llg/file22ab773487ab.shp"
templaz()
#> [1] "/tmp/RtmpQD0Llg/file22ab60e93bc6.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpqlo3b4/file25237d15b6b6.shp"
templaz()
#> [1] "/tmp/Rtmpqlo3b4/file252354f92a9e.laz"
```

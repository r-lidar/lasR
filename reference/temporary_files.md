# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpAE6a4F/file22722903f053.shp"
templaz()
#> [1] "/tmp/RtmpAE6a4F/file227231cf32cb.laz"
```

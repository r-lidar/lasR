# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmp4Fagql/file2539382be4bc.shp"
templaz()
#> [1] "/tmp/Rtmp4Fagql/file25396446b0c3.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmpy55V7f/file254932b07a4f.shp"
templaz()
#> [1] "/tmp/Rtmpy55V7f/file254929aed085.laz"
```

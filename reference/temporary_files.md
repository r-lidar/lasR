# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpP5nIen/file240857b1138d.shp"
templaz()
#> [1] "/tmp/RtmpP5nIen/file24087326c1f0.laz"
```

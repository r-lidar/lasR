# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpAKasvg/file210e553c99a7.shp"
templaz()
#> [1] "/tmp/RtmpAKasvg/file210e36143ab2.laz"
```

# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmp35k4wL/file22611e4bc3da.shp"
templaz()
#> [1] "/tmp/Rtmp35k4wL/file22614c2dc64.laz"
```

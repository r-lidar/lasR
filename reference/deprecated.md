# Deprecated

Use [reader](https://r-lidar.github.io/lasR/reference/reader.md)
instead.

## Usage

``` r
reader_las(filter = "", select = "*", copc_depth = NULL, ...)

reader_las_coverage(filter = "", select = "*", copc_depth = NULL, ...)

reader_las_circles(
  xc,
  yc,
  r,
  filter = "",
  select = "*",
  copc_depth = NULL,
  ...
)

reader_las_rectangles(
  xmin,
  ymin,
  xmax,
  ymax,
  filter = "",
  select = "*",
  copc_depth = NULL,
  ...
)
```
